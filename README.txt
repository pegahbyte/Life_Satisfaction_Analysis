This project was part of the course "Bayesian Machine Learning" (MSc Data Science). The code can be obtained from the jupyter notebook.

For a more comprehensive and non-technical overview, see the corresponding paper (pdf).

The goal of this analysis is to find and select the most important factors that are correlated to self-reported life-satisfaction. 
To do so a regression analysis and a lasso regression analysis is conducted. Moreover, the data is clustered and re-analysed on cluster level with interesting findings.
